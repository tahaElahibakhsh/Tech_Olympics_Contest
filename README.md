### Evaluation Metric

Performance is measured using **RMSE (Root Mean Squared Error)**, converted into a nonlinear score:

```
Score = 100 Ã— exp(- RMSE / std(Y_true))
```

* **Score = 100** â†’ Perfect predictions
* Lower RMSE relative to natural data variance earns higher scores

During the competition:

* Public score = evaluation on **30%** of test data
* Final score = evaluation on remaining **70%** (post-competition)

This setup prevents overfitting and rewards generalizable models.
